## 🏠 简体中文

### 项目简介

**卡塔藏译 - 喜马拉雅编辑器** 是一款高性能、支持快捷键映射的藏文网页版富文本编辑器。它特别针对 iPad 外接键盘用户进行了优化，内置了经典的 **喜马拉雅 (Himalayan)** 键盘映射，让你无需在系统层安装繁琐的输入法，即可在浏览器中获得流畅的藏文输入体验。

### 🚀 核心特性

- 🏔️ **喜马拉雅映射**：完美支持经典喜马拉雅键盘映射逻辑。
- ⌨️ **iPad 优化**：深度兼容 iPad 外接蓝牙键盘及妙控键盘。
- 📝 **富文本编辑**：支持加粗、斜体、下划线、标题、列表、图片插入。
- 📄 **Word 导出**：一键将编辑内容导出为标准 `.docx` 格式。
- 🎨 **极简设计**：基于 Tailwind CSS 构建的现代、响应式 UI。
- ⚡ **即时切换**：使用 `Shift + K` 快速在喜马拉雅模式与系统常规输入之间切换。

### 🛠️ 技术栈

- **框架**: [React 19](https://react.dev/)
- **构建**: [Vite 6](https://vite.dev/)
- **编辑器**: [Tiptap](https://tiptap.dev/)
- **样式**: [Tailwind CSS 4](https://tailwindcss.com/)

### 📦 快速开始

```bash
git clone https://github.com/yourusername/kata-editor.git
cd kata-editor
npm install
npm run dev
```

## 🌍 English

### Introduction

**Kata Himalayan Editor** is a high-performance Web-based rich-text editor for Tibetan. It is specifically optimized for iPad external keyboards, featuring a built-in **Himalayan** keyboard mapping, allowing users to type Tibetan fluently in any browser without installing native input methods.

### 🚀 Features

- 🏔️ **Himalayan Mapping**: Fully supports the classic Himalayan keyboard logic.
- ⌨️ **iPad Optimized**: Deep compatibility with iPad external Bluetooth and Magic Keyboards.
- 📝 **Rich Text**: Bold, Italic, Underline, Headings, Lists, and Image support.
- 📄 **Word Export**: Export content to standard `.docx` files with one click.
- 🎨 **Minimalist Design**: Modern, responsive UI built with Tailwind CSS.
- ⚡ **Instant Switch**: Use `Shift + K` to toggle between Himalayan mode and system input.

### 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build**: [Vite 6](https://vite.dev/)
- **Editor Engine**: [Tiptap](https://tiptap.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)

### 📦 Quick Start

```bash
git clone https://github.com/yourusername/kata-editor.git
cd kata-editor
npm install
npm run dev
```
